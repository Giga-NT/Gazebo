# 📱 Как установить APK на телефон

## ✅ APK файл готов!

**Путь к файлу:**
```
C:\Users\Пользователь\777\FrameConstructor\android\app\build\outputs\apk\debug\app-debug.apk
```

**Размер:** 75.7 MB

---

## 📤 Способ 1: Через USB кабель

1. **Подключите телефон к ПК** через USB
2. **На телефоне** выберите "Передача файлов" (MTP)
3. **Скопируйте** `app-debug.apk` на телефон (в любую папку, например Download)
4. **Отключите телефон** от ПК
5. **На телефоне:**
   - Откройте "Файлы" или "Проводник"
   - Найдите `app-debug.apk`
   - Нажмите на файл
   - Нажмите "Установить"
   - Разрешите установку из неизвестных источников (если спросит)
   - Готово! 🎉

---

## 📤 Способ 2: Через Google Drive / Облако

1. **Загрузите APK** в Google Drive / Dropbox / Яндекс.Диск
2. **На телефоне** откройте приложение облака
3. **Скачайте** APK файл
4. **Откройте** скачанный файл
5. **Установите** приложение

---

## 📤 Способ 3: Через Telegram

1. **Откройте Telegram** на ПК
2. **Отправьте** `app-debug.apk` в "Избранное" (Saved Messages)
3. **На телефоне** откройте Telegram
4. **Скачайте** файл
5. **Откройте** и установите

---

## 📤 Способ 4: Через email

1. **Прикрепите** APK к письму
2. **Отправьте** себе на email
3. **На телефоне** откройте письмо
4. **Скачайте** вложение
5. **Установите** приложение

---

## ⚠️ Если телефон не устанавливает приложение

### Ошибка: "Приложение не установлено"

1. **Настройки** → **Безопасность** → **Неизвестные источники**
2. **Разрешите** установку из неизвестных источников
3. **Попробуйте снова**

### Ошибка: "Конфликт подписей"

Если уже установлена версия приложения:
1. **Удалите** старую версию
2. **Установите** новую

### Ошибка: "Недостаточно памяти"

1. **Очистите место** на телефоне
2. **Попробуйте снова**

---

## 🔧 Настройка телефона для разработки (опционально)

### Включить отладку по USB:

1. **Настройки** → **О телефоне**
2. **7 раз нажмите** на "Номер сборки"
3. **Настройки** → **Для разработчиков**
4. **Включите** "Отладка по USB"
5. **Подключите** телефон к ПК
6. **Разрешите** отладку на телефоне

### Установка через ADB:

```bash
adb install app-debug.apk
```

---

## 📞 Приложение установлено!

После установки:
- Найдите иконку **"Giga Конструктор"** на рабочем столе
- Откройте и проверьте работу
- 3D модели должны работать корректно

---

## 🔄 Обновление приложения

Для обновления:
1. **Соберите новый APK** (как описано выше)
2. **Установите** поверх старой версии
3. **Данные сохранятся**

Или удалите старую версию и установите новую.
