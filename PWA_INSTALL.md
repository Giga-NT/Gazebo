# PWA Installation Guide

## Как установить приложение

### На Android (Chrome)

1. Откройте сайт в браузере Chrome
2. Нажмите на меню (три точки в правом верхнем углу)
3. Выберите **"Установить приложение"** или **"Добавить на главный экран"**
4. Подтвердите установку
5. Приложение появится на главном экране

### На iOS (Safari)

1. Откройте сайт в браузере Safari
2. Нажмите кнопку **"Поделиться"** (квадрат со стрелкой вверх)
3. Прокрутите вниз и выберите **"На экран 'Домой'"**
4. Введите название (или оставьте по умолчанию)
5. Нажмите **"Добавить"** в правом верхнем углу
6. Приложение появится на главном экране

### На компьютере (Chrome/Edge)

1. Откройте сайт в браузере Chrome или Edge
2. В адресной строке появится значок установки (⬇️ или 📥)
3. Нажмите на него
4. Выберите **"Установить"**
5. Приложение появится на рабочем столе и в меню "Пуск"

## Преимущества установленного приложения

- ✅ Быстрый запуск с главного экрана
- ✅ Работает офлайн (кэширование данных)
- ✅ Нет адресной строки браузера
- ✅ Выглядит как нативное приложение
- ✅ Автоматические обновления

## Возможности приложения

- 3D конфигуратор навесов, теплиц и беседок
- Расчёт стоимости и материалов
- Сохранение проектов в личном кабинете
- Экспорт и печать смет

## Решение проблем

### Приложение не устанавливается

1. Убедитесь, что используете HTTPS соединение
2. Проверьте, что браузер поддерживает PWA (Chrome, Safari, Edge, Firefox)
3. Очистите кэш браузера и попробуйте снова

### Приложение не работает офлайн

1. Откройте приложение хотя бы один раз онлайн
2. Дождитесь загрузки service worker
3. Проверьте соединение

### Не приходит уведомление об установке

1. Нажмите кнопку "Установить" в приложении (если доступна)
2. Используйте меню браузера для установки
