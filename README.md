# Giga FrameConstructor

**Интерактивный 3D-конфигуратор для производителей и покупателей навесов, теплиц и беседок**

## Возможности
- 🏗️ **3D-визуализация** в реальном времени
- 📐 **Изменяемые размеры** (длина, ширина, высота)
- 🎨 **Выбор материалов** (поликарбонат, металл и др.)
- 💰 **Автоматический подсчет стоимости** при изменении параметров
- 📱 **Адаптивный интерфейс** (десктоп + мобильные)

## Для кого
- **Производители**: покажите клиенту точный результат до производства
- **Покупатели**: детально рассмотрите будущее изделие, подберите параметры
